CharityBid: premium real-time auction platform where users bid on celebrity meetings/dinners, 100% proceeds to charity.

**Why:** Quick MVP build — designed for a short development cycle with AI tooling assistance.

**How to apply:** All implementation decisions should reference prd.md. The blueprint covers schema, functions, RLS, Stripe lifecycle, component hierarchy, data flow, and build phases. Follow it section by section. Auctions are admin-seeded via SQL for MVP (no admin dashboard).
